# vps-file-system
A simple vps file system with .wav and .mp4 file uploads with an ssh client.

A simple node js program which will be installed in vps.

media file upload and delete. using solar puTTy or puTTY 

steps 1. Create node js program

step 2. I have to install node js project in vps with bash commands (with gist)

step 3. Cli system in the program in the vps with bash commands.

Step 4. Create & delete a file and use postman to check.

Step 5. I have to be able to upload and delete a .wav and .mp4 file in the with an api in the node project inside the vps.

step 6. An API to delete and update the files.
